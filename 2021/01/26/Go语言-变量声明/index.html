Go语言-变量声明

本文最后更新于：2021年2月9日 凌晨

变量声明

导引

package main

import "fmt"

func main(){
	fmt.Println("Hello World!")
}

注意

  1. 所有的.go文件都需要有package声明。(package main最后会编译成一个可执行文件)
  2. import为导入语句,导入各种包
  3. func main() 为程序入口
  4. 函数外只能放置标识符(变量、常量、函数、类型)的声明

变量和常量

标识符

由程序员定义的具有特殊意义的词,比如变量名、常量名、函数名等等。GO语言中标识符只能由字母、数字和下划线组成,并且只能以字母和下划线开头

关键字

具有特殊含义的标识符。
Go语言中有25个关键字,另还有37个保留字。网上都可以找到,遂不在此陈列。

变量

Go变量需要先声明再使用

声明变量

var s1 string 声明一个保存字符串类型的名字叫s1的变量

其他同理，这种称作标准声明

批量声明

var(
	name1 string
    age1 int
    marriage bool
)

变量的初始化

go语言中声明变量必须使用，不使用就编译不过去

func main(){
    name1 = "lily"
    age1 = 14
    marriage = false
    
    fmt.Println(age1) //ln代表换行
    fmt.Printf("marriage:%s",marriage) //f 即format
}

PS：Go语言编程代码格式规范推荐 ‘驼峰式命名’

驼峰式命名：（根据Java自行命名）

  1. 包名：多单词组成时所有字母都小写
  2. 类名、接口名：多单词组成时，所有单词的首字母大写（大驼峰）
  3. 变量名、方法名：多单词组成时，第一个单词首字母小写，第二个单词开始，每个单词首字母大写（小驼峰）
  4. 常量名：所有字母都大写，多单词时每个单此用下划线链接

PS：更多细节见《代码整洁之道》

声明变量同时赋值

  1. 第一种

    var name2 = "Jack"
var age2 = 50

  2. 第二种

    var name3,age3 = "Pony",45 //批量声明批量赋值

类型推导

有时候将变量的类型省略，编译器会根据等号右边的值来推导变量的类型完成初始化 

var name4 = "Sunsan"
var age4 = 16

短变量声明（简短模式）

函数内部使用:=

package main

import "fmt"

func main(){
    
    age5 := 17 //适用于局部变量声明
    
    fmt.Println(age5)
}

注意：

  1. 简短模式定义变量同时必须显式初始化
  2. 不能提供数据类型
  3. 只能用在函数内部

匿名变量

使用批量赋值时，若是想要忽略某个值，可以使用匿名变量（anonymous variable）。匿名变量用一个_表示：

func foo()(int,string){
    return 10,"s3"
}

func main(){
    x,_ := foo()
    _,y := foo()
    fmt.Println("x=",x)
    fmt.Println("y=",y)
}

匿名变量不占用命名空间，不会分配内存，所以匿名变量之间不存在重复声明——_多用于占位，表示忽略值
