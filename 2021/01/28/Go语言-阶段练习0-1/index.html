阶段练习

素数练习：打印200-1000的素数

解题思路：

素数：指在大于1的自然数中，除了1和它本身以外不再有其他因数的自然数。

  1. 首先必然要遍历200-1000的所有整数，那么就肯定要用for循环

  2. 其次，遍历了之后如何确定这个数是否是素数呢？关键思路就是在2到这个数-1之间不能被整除。

  3. 按照这个思路，我们进一步进行简化：不必对2—这个数-1的每个数都去做除法，只需要在2—根号这个数之间做除法即可。因为如果一个数可以被整除，且其中一个因子比开方后的数大，那么对应地就一定会有一个比开方后小的因子。（如16，2和8是其因子，8比4大，2比4小）

基本解题思路有了，代码如下：

//1.打印200-1000的素数
count := 0                    //设置计数器
for i := 200; i < 1000; i++ { //遍历200~1000之间的所有整数
	flag := true
	for j := 2; j < i/2; j++ { //对i是否是素数进行判断提供除数
		if i%j == 0 {
			//证明i就不是素数
			flag = false
			break
		}
	}
	//如果内层循环没有break，则证明i是素数
	if flag {
		fmt.Printf("%d\t", i)
		count++
		if count%5 == 0 {
			fmt.Println() //如果一行满五个则换行
		}
	}
}
fmt.Println()
fmt.Printf("200~1000之间共计有素数%d个", count)

打印九九乘法表

解题思路：

  1. 双层for循环
  2. 内层循环中用带格式的输出语句输出
//2.打印九九乘法表
for i := 1; i <= 9; i++ { //外层for循环主要表示乘数
	for j := 1; j <= i; j++ { //内层for循环主要表示被乘数
		fmt.Printf("%d * %d = %d  ", j, i, i*j)
	}
	fmt.Println()
}
